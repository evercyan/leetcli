## [22. 括号生成](https://leetcode-cn.com/problems/generate-parentheses/)

---

> 难度: 中等

> 分类:  [字符串](https://leetcode-cn.com/tag/string/)  [回溯算法](https://leetcode-cn.com/tag/backtracking/) 

---

##### 题目描述

```
给出 n 代表生成括号的对数，请你写出一个函数，使其能够生成所有可能的并且有效的括号组合。

例如，给出 n = 3，生成结果为：

[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]


```
